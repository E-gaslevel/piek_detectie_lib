# Peakdetect

Lightweight C project for peak detection.

## Build

```sh
mkdir -p build && cd build
cmake ..
cmake --build .
```

## Run example

After building:

```sh
./examples/peak_example
```

## Tests

Run the tests via CTest from the build directory:

```sh
cd build
ctest -V
```
